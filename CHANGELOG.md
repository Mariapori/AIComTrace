# Changelog

All notable changes to AIComTrace will be documented in this file.

## [0.1.0] - 2026-04-24

### Added
- Initial release
- Status bar toggle for AI co-author mode
- Quick Pick menu for selecting AI tools
- Built-in support for GitHub Copilot, Claude, ChatGPT, Gemini, Cursor, and Windsurf
- Custom AI tool configuration
- Auto-append Co-authored-by trailer to SCM input box
- Dedicated "Commit with AI Co-Author" command
- Remove trailer command
- Persistent state across sessions
- Extension settings for default tool, custom tools, and auto-append behavior
